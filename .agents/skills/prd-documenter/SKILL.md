# PRD Documenter Skill

## Goal
To provide consistent, high-quality documentation for the TikTak platform.

## Instructions
1. **Analyze**: Scan the workspace to ensure alignment with existing Vite and Firebase code.
2. **Draft**: Create a PRD focusing on Problem Statement, User Stories, and Success Metrics.
3. **Architect**: Map requirements to specific Vite components and Firebase services.

## Constraints
- Keep logic atomic; do not include non-incidents features.
