# Feature Architect Skill

## Instructions
1. **Gap Analysis**: Compare the new requirement against the existing `docs/architecture.md`.
2. **Draft PRD**: Focus on how the feature integrates into the current investigation flow.
3. **Tech Spec**: Define required Firebase Function changes or new Tailwind components.

## Output
Create files in `docs/PRDS/` and update `docs/architecture.md` if the system design changes.
